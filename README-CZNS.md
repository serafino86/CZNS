# CZNS Alliance Hub 🎯

**Professional Alliance Management System for Last War: Survival**

Powered by NGC Alliance Hub Technology  
Customized for CZNS Alliance

---

## 🌟 Features

### 🎮 War Room Defense Tracker
- Real-time power tracking with 3D visualization
- Individual player power monitoring
- Alliance-wide statistics dashboard
- Multi-language support (EN/IT/FR/DE)

### 🗺️ Alliance Strategy Manager
- **Canyon Mode**: 12-building tactical coordination
- **Desert Storm Mode**: 11-building strategic planning
- Team A/B separate management
- Load Last Battle feature with smart matching
- Kill Swat and Reserves tracking
- Building-specific assignments
- Formation history with automatic save

### 📊 Personal Stats Dashboard
- Total Power tracking with trend analysis
- Squad Details breakdown (Missiles, Aircraft, Tanks)
- Growth visualization with dual chart system
- Historical data tracking

---

## 🚀 Quick Start for CZNS Alliance

1. **R4/R5**: Get alliance code from leadership
2. **Visit**: Alliance hub URL (provided by R4/R5)
3. **Enter code**: Submit your power data
4. **Track progress**: View your personal dashboard
